Lista - Conceito

Conceito

Uma lista encadeada é uma representação de uma sequência de objetos, todos do mesmo tipo, como feito na memória RAM (= random access memory) do computador.

Ela é composta por células que apontam para o próximo elemento da lista. Para "ter" uma lista, basta guardar seu primeiro elemento, e seu último elemento aponta para uma célula nula.

Cada elemento da sequência é armazenado em uma célula da lista: o primeiro elemento na primeira célula, o segundo na segunda e assim por diante.

Segue abaixo um pseudocódigo
Nó de registro [      dados // O dado a ser armazenado no nó      próximo // Referência ao próximo nó; nulo para o último nó   ]   Lista de registros [       nó primironó // ponteiro para o primeiro nó da lista; null para uma lista       vazio   ] A lista é simples e fácil de abrir primeiro e depois de cada próximo até fim.   nó: = list.firstnó   enquanto nó não nulo [       nó: = nó.next   ]
