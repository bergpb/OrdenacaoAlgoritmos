Pilha - Conceito

Conceito

Uma pilha é uma estrutura de dados que admite remoção de elementos e inserção de novos objetos.

Mais especificamente, uma pilha (= stack) é uma estrutura sujeita à seguinte regra de operação: sempre que houver uma remoção, o elemento removido é o que está na estrutura há menos tempo.

Em outras palavras, o primeiro objeto a ser inserido na pilha é o último a ser removido. Essa política é conhecida pela sigla LIFO (= Last-In-First-Out).

Segue abaixo um pseudocódigo
registro Nodo      dados      próximo      Pilha de discos       Node stackPointer      função push (Stack stack, Element element)       novo (newNode)       newNode.data: = elemento       newNode.next: = stack.stackPointer       stack.stackPointer: = newNode     função pop (pilha Stack)       nó: = stack.stackPointer       stack.stackPointer: = node.next       elemento: = node.data       elemento de retorno     função top (pilha Stack)       return stack.stackPointer.data     comprimento da função (pilha de pilha)       comprimento: = 0       nó: = stack.stackPointer       enquanto nó não nulo           comprimento: = comprimento + 1           nó: = node.next             retorna tamanho;
